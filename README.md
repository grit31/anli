# 机器学习练习项目

这个仓库是我平时学习和练习机器学习相关内容的地方，主要包括PyTorch、TensorFlow和强化学习的一些代码。

## 目录结构

- `pytorch/` - PyTorch相关的练习代码和数据
- `tensorflow/` - TensorFlow的一些实验，主要是图像分类和回归
- `rl/` - 强化学习的学习资料和代码实现
- `课件/` - 收集的一些课程资料
- `demo*.py` - 各种小demo，测试用的

## 说明

代码比较杂乱，主要是自己学习用的，有些是跟着教程写的，有些是自己瞎折腾的。

大文件用了Git LFS存储，clone的时候注意一下。

## 环境

主要用Python 3.x，具体依赖看各个子目录里的
